# 贪吃蛇大作战游戏设计方案

## 选定设计方案：现代竞技游戏风格

### 设计哲学
采用**现代竞技游戏美学**，融合霓虹灯赛博朋克元素与流畅的交互体验。设计强调速度感、竞争性和沉浸感，通过高对比度、动态效果和精准的反馈机制来增强玩家的游戏体验。

### 核心设计原则
1. **高能量视觉**：采用深色背景搭配鲜艳的霓虹色彩，营造紧张刺激的竞技氛围
2. **即时反馈**：每个操作都有明确的视觉和音效反馈，增强游戏的沉浸感
3. **清晰的信息层级**：实时显示玩家得分、排名、对手信息，让玩家时刻掌握游戏状态
4. **流畅的动画**：蛇的移动、食物生成、碰撞效果都采用平滑的过渡动画

### 色彩哲学
- **主背景**：深灰蓝色（`#0f1419`），营造深邃的竞技场氛围
- **玩家蛇**：霓虹绿（`#00ff88`），代表玩家的主动性和活力
- **AI 蛇**：霓虹紫（`#ff00ff`）、霓虹蓝（`#00ffff`）、霓虹橙（`#ff6600`），区分不同对手
- **食物**：霓虹黄（`#ffff00`），吸引注意力
- **UI 元素**：半透明白色文字和边框，保持清晰的对比度

### 布局范式
- **居中竞技场**：Canvas 游戏区域居中显示，四周环绕 UI 信息面板
- **顶部信息栏**：显示玩家得分、排名、游戏时间
- **右侧对手面板**：实时显示所有 AI 对手的信息（长度、得分、状态）
- **底部控制栏**：游戏状态指示、暂停/继续按钮、重新开始按钮

### 标志性元素
1. **网格背景**：淡色网格覆盖整个游戏区域，增强科技感
2. **蛇身发光效果**：蛇身周围有淡色光晕，突出蛇的存在感
3. **碰撞爆炸效果**：蛇碰撞时产生粒子爆炸动画
4. **食物脉动效果**：食物持续脉动，吸引玩家注意

### 交互哲学
- **键盘控制**：方向键或 WASD 控制蛇的移动，反应灵敏
- **触摸控制**：移动设备上支持滑动或虚拟方向键
- **实时反馈**：蛇吃到食物立即产生视觉反馈和得分增加
- **游戏结束动画**：蛇消失时产生爆炸效果，显示最终排名

### 动画指南
- **蛇移动**：平滑的线性插值，每个网格单位的移动时间为 100ms
- **食物出现**：从无到有的淡入效果（300ms）
- **碰撞爆炸**：快速的粒子扩散动画（500ms），伴随屏幕微震
- **UI 更新**：数字变化时的轻微放大效果（200ms）
- **游戏暂停**：整个游戏区域的模糊效果和半透明遮罩

### 排版系统
- **标题字体**：`Orbitron`（谷歌字体）- 科技感十足的等宽字体
- **正文字体**：`Roboto Mono`（谷歌字体）- 清晰易读的等宽字体
- **标题层级**：
  - 大标题（H1）：`Orbitron` 32px 粗体，用于游戏标题
  - 中标题（H2）：`Orbitron` 20px 粗体，用于面板标题
  - 小标题（H3）：`Roboto Mono` 14px 粗体，用于信息标签
  - 正文（P）：`Roboto Mono` 12px 常规，用于数值显示

## 游戏功能规划

### 核心游戏机制
1. **玩家蛇**：由玩家通过键盘控制，初始长度为 3 格
2. **AI 蛇**：3-5 条 AI 对手，采用简单的寻食 AI 算法
3. **食物生成**：随机生成食物，蛇吃到食物后长度增加 1
4. **碰撞检测**：
   - 蛇与自身碰撞 → 玩家失败
   - 蛇与其他蛇碰撞 → 碰撞的蛇失败
   - 蛇与边界碰撞 → 蛇失败
5. **排名系统**：实时计算所有蛇的排名，基于长度和得分

### UI 组件
1. **游戏标题**：顶部中央显示 "SNAKE BATTLE"
2. **玩家信息**：显示玩家得分、排名、蛇长度
3. **对手信息**：显示所有 AI 对手的实时信息
4. **游戏状态**：显示游戏是否运行、暂停或结束
5. **控制按钮**：暂停/继续、重新开始、返回主菜单

### 游戏流程
1. **开始界面**：显示游戏规则和开始按钮
2. **游戏进行**：玩家控制蛇，AI 蛇自动移动
3. **游戏暂停**：按空格暂停，显示暂停菜单
4. **游戏结束**：显示最终排名和重新开始选项
